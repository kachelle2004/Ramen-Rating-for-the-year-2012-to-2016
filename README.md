# Ramen-Rating-for-the-year-2012-to-2016
This dataset shows the popular Ramen brands and flavours across countries such as Japan, Thailand, Singapore, and many more. It identifies the popular brands that was listed in the top ten for the years 2012 to 2016.

This dataset was taken from Kaggle's dataset. The following link is stated below: https://www.kaggle.com/datasets/residentmario/ramen-ratings

In This Project:
1) Cleaned up missing and misvalued data from the dataset.
2) Identified the popular brand and its origin of the Ramen
3) There are a few types of ramen that have a rating of 5 but were not featured on the top ten popular Ramen in each year.
4) Some of the Top Ten Rating Ramen have appeared more than consecutive years, or in 1 year, have a few types of variety appear in the Top Ten Rating.

Tools and Programme:
This dataset was performed in Jupiter Notebook, in particular, the programming language Python and libraries such as Pandas, Numpy to analyse and clean the data, and  using  matplotlib to plot some bar graphs to display the relationship between the Ramen variety and brands.
